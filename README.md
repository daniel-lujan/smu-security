## Quick start

First, clone the repository:

```bash
git clone https://github.com/daniel-lujan/smu-security.git
```

Enter the project:

```bash
cd smu-security
```

Install dependencies:

> [!NOTE]
> You can use alternative package managers like `pnpm` or `bun`.

```bash
npm install
```

Run development server:

```bash
npm run dev
```
